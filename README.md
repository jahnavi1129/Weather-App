🌦️ Weather App
📌 Project Overview

Weather App is a responsive web application developed using HTML, CSS, and JavaScript. It allows users to search for weather information by entering a city name. The application fetches real-time weather data from the OpenWeatherMap API and displays weather details in an interactive user interface.

🚀 Features
Search weather by city name
Display current temperature
Show weather conditions
Display humidity and wind speed
Error handling for invalid city names
Responsive design for all devices

🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6+)
Fetch API
Async/Await
OpenWeatherMap API
